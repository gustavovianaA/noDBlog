Conceitos de Computação - Armazenamento de Dados

Armazenamento de Dados

Toda informação que é armazenada e processada por um computador é codificada em uma cadeia de 0s e 1s.Esses digitos recebem o nome de binary digits ou bits . Isso é válido para qualquer arquivo ou programa.

Memória Principal (RAM)

A memória principal é dividida em células, sendo que cada célula pode armazenar oito bits, que equivale à um byte . Para que seja possível identificar essas células, cada uma recebe um nome, que chamamos de endereço de memória . Cada uma dessas células pode ser acessada de maneira aleatória conforme for necessário, pois todas possuem um endereço. Daí surge o nome RAM ( Random Acess Memory )

Bits na memória RAM são armazenados através de cargas elétricas. A memória RAM é volátil, as informações contidas nela são apagadas quando o computador é desligado. Para que seja possível a persistência de dados são utilizados dispositivos de armazenamento em massa.

Armazenamento em Massa

Todos os dispositivos de armazenamento em massa armazenam bits. O que muda é a forma com que esses bits são armazenados. Em um HD os eletroímãs presentes nos disco rígido podem ser modificados afim de indicar um bit de valor 0 ou 1. Um SSD armazena bits através de seus circuitos eletrônicos.
